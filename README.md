# Tron Light Cycles

A two-player real-time light cycle battle game inspired by the 
Tron movie, built in Java with Swing and SQLite. Solo university project.

## About the Game
Two players race light motorcycles on a grid, each leaving a 
glowing trail behind. A player loses if they hit the boundary 
or ride into either player's trail. The last one standing wins.

## Features
- Two-player simultaneous input (WASD + Arrow Keys)
- Custom player names and trail color selection before each game
- Persistent win tracking via SQLite database
- High score table showing top 10 players
- Restart option from the menu
- Full menu system (New Game, High Scores, Restart)

## Tech Stack
- **Language:** Java
- **GUI:** Java Swing
- **Database:** SQLite (via JDBC)
- **Type:** Solo project
